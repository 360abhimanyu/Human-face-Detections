# 🤖 AI Face Detection Dashboard (Azure Computer Vision)

## 📌 Overview
This project is a modern web-based dashboard that detects human faces in uploaded images using Microsoft Azure Computer Vision API.  
Users can securely provide their own API Key and Endpoint, upload images, and visualize detected faces with bounding boxes in real time.

---

## 🚀 Features
- 🔐 User-provided Azure Key & Endpoint
- 🖼️ Image upload with preview
- 🧠 AI-powered face detection
- 📦 Bounding box visualization on faces
- 🎨 Modern UI with glassmorphism design
- ⚡ Fast and responsive experience

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (Glass UI + Responsive Design)
- JavaScript (Vanilla JS)
- Microsoft Azure Computer Vision API

---

## ⚙️ How It Works
1. User enters:
   - Azure Endpoint  
   - Azure API Key  
2. Uploads an image  
3. The app sends the image to Azure Computer Vision API  
4. Faces are detected and highlighted on the image  

---

## 🔑 Azure Setup
1. Go to Azure Portal  
2. Create a **Computer Vision Resource**  
3. Copy:
   - Endpoint → `https://<your-region>.api.cognitive.microsoft.com`
   - Key
